# Счастливый Марик — 3D MVP

Интерактивный 3D-кот, которого можно крутить, гладить и тыкать.

- Поворот камеры — **ЛКМ + мышь / палец**
- Зум — **колёсико мыши / pinch на телефоне (через масштаб страницы)**
- Глажение — **двигать курсор/палец по коту**
- Клик/тап по коту — **ругается** `!!!###!!!???!!!`

Сделано на чистом **three.js**, без сборщиков и зависимостей.

## Локальный запуск

Просто открой `index.html` в браузере (Chrome, Firefox, Edge).

## Деплой на GitHub Pages

1. Создай пустой репозиторий на GitHub, например `happy-marik`.
2. Залей в него файлы:
   - `index.html`
   - `README.md`
   - `.nojekyll`
3. В настройках репозитория:
   - Зайди в **Settings → Pages**.
   - В разделе **Build and deployment** выбери:
     - Source: **Deploy from a branch**
     - Branch: `main` / `master` (какой у тебя) и папку **/ (root)**.
4. Сохрани настройки. Через пару минут страница будет доступна по адресу:

`https://<твой-логин>.github.io/<имя-репозитория>/`
